Clue Simulator

Clue Simulator is a digital implementation of the classic mystery board game Clue (Cluedo).
The game challenges players to solve a murder mystery by identifying the suspect, weapon, and room involved in the crime.

This project simulates the core mechanics of the Clue game including card distribution, turn-based gameplay, suggestions, and final accusations.


Features

Randomized suspect, weapon, and room selection

Automatic card distribution to players

Turn-based gameplay system

Suggestion and accusation mechanics

Logical deduction system

Interactive console-based interface (or GUI if applicable)

Game result tracking

Game Components

Suspects: Miss Scarlet, Colonel Mustard, Professor Plum, etc.

Weapons: Knife, Revolver, Rope, etc.

Rooms: Kitchen, Ballroom, Library, etc.

Future Improvements

Add AI-based intelligent opponents

Multiplayer support

Improved GUI

Save/Load game feature

Player statistics dashboard
